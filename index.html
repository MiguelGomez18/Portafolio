EliteBle

Descripción:

Aplicación desarrollada para Elite Flower que permite la comunicación por Bluetooth Low Energy (BLE) con dispositivos ESP32 para realizar configuraciones remotas. Incluye autenticación consultando SQL Server para el inicio de sesión y flujos de configuración de dispositivos.

Tecnologías:

  • Flutter
  • Dart
  • Bluetooth BLE
  • SQL Server

SQL

Descripción:

Aplicación para ingresar parámetros de conexión (servidor, credenciales, base de datos) y visualizar tablas y vistas de la base de datos en el teléfono. Permite ejecutar consultas manuales y visualizar datos en tablas desde el móvil.

Tecnologías:

  • Flutter
  • Dart
  • Conexión a bases de datos remotas

FlowDay

Descripción:

Aplicación personal para gestión de tareas con vista semanal/mensual, recordatorios mediante notificaciones y múltiples espacios (trabajo, casa, estudio, etc.). Guarda los datos localmente usando JSON en SharedPreferences para preservar la privacidad del usuario. (En construcción: ~90%).

Tecnologías:

  • Flutter
  • Dart
  • SharedPreferences (JSON)
  • Notificaciones locales

FlowPay

Descripción:

Aplicación personal para gestionar ingresos y categorías de gasto. Permite registrar transacciones, ver informes visuales (gráficas de línea y circulares), establecer objetivos y registrar préstamos. Guarda los datos localmente para garantizar privacidad. (En construcción: ~60%).

Tecnologías:

  • Flutter
  • Dart
  • SharedPreferences (JSON)
  • Gráficos (librería integrada)

Ventrix

Descripción:

Aplicación web desarrollada como proyecto final en SENA para gestión posventa. Permite gestionar pedidos y flujos según roles (mesero, cajero, cocinero, administrador de sucursal y propietario), con control de permisos y organización por sucursales y mesas.

Tecnologías:

  • Vue 3
  • Java (Spring Boot)
  • Docker
  • Despliegue: Railway
